Project Description

This co-op multiplayer puzzle platformer was one of my first big projects. It features a level editor, networked multiplayer, a few dozen tile types / objects, dynamic lights, and portals. I stopped working on it because the code became very cluttered and redundant as I was still learning to program at the time.
