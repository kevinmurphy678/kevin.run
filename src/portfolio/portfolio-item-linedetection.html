Project Description *WIP*

This project uses opencv to detect lines on a race track in a game. It sends controls to the game (currently rFactor) and attempts to keep the car on the track. It uses a virtual webcam to monitor the game and calculate line detection in real time. I hope to improve the detection and try it with other games sometime.

Made With

  • emgucv (openCV for C# / .NET)
  • Manycam(virtual webcam software)
